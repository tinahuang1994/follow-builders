## What this fork adds
- **Cheaper X source** — swaps the official X API for a pay-per-use provider (~30× cheaper).
- **腾讯研究院 AI速递** — adds a Chinese-language source.
- **小白版 HTML deck output** — delivers a visual, beginner-friendly slide deck
  (instead of a text digest), explaining each item so a non-technical reader gets it.
- **Personalized "So what · for you"** — asks the user what they do/are building and
  tailors the takeaway on every item to them.

The original source list, feed plumbing, and prompt structure remain Zara's work.
